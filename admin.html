How it works ✦
Everything you create here is saved in this Admin. When ready, click Download koino.html — it generates a complete updated app with all your questions baked in. Replace the old file wherever you host it.
1
Build your content in Admin
Add questions, create editions, bulk paste

Use Bulk Add or the Questions page to add all your cards. Create as many categories and editions as you need.

✓ Your content auto-saves every time you make a change.
2
Click Download koino.html
Generates a complete updated app file

The downloaded file is the complete Koino app with all your current content baked in. One file, no dependencies.

📁 Downloads/
  └── koino.html ← your new app file
3
Replace the old file
Wherever your app is hosted
GitHub Pages: upload as index.html ✓
Netlify Drop: drag & drop ✓
Any host: overwrite the old file ✓

The URL stays the same. Only the content updates.

4
Back up your data
Save a JSON backup if you change devices

Admin data is saved in your browser. If you switch devices, use the backup tools below.
